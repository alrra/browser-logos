How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/samsung-internet-gear-vr`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/samsung-internet-gear-vr`

* Use [`cdnjs`][cdnjs].
