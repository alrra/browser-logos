How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/edge-canary`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/edge-canary`

* Use [`cdnjs`][cdnjs].
