How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/opera-touch`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/opera-touch`

* Use [`cdnjs`][cdnjs].
