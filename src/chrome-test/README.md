How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/chrome-test`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/chrome-test`

* Use [`cdnjs`][cdnjs].
