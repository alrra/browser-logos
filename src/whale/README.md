How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/whale`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/whale`

* Use [`cdnjs`][cdnjs].
