How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/firefox-preview-nightly`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/firefox-preview-nightly`

* Use [`cdnjs`][cdnjs].
