How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/servo`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/servo`

* Use [`cdnjs`][cdnjs].
