How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/firefox-preview-beta`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/firefox-preview-beta`

* Use [`cdnjs`][cdnjs].
