How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/samsung-internet_5`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/samsung-internet_5`

* Use [`cdnjs`][cdnjs].
