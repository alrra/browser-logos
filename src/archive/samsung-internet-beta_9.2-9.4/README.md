How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/samsung-internet-beta_9.2-9.4`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/samsung-internet-beta_9.2-9.4`

* Use [`cdnjs`][cdnjs].
