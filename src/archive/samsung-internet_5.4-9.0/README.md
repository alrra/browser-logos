How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/samsung-internet_5.4-9.0`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/samsung-internet_5.4-9.0`

* Use [`cdnjs`][cdnjs].
