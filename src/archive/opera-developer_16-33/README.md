How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/opera-developer_16-33`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/opera-developer_16-33`

* Use [`cdnjs`][cdnjs].
