How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/safari-ios_1-6`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/safari-ios_1-6`

* Use [`cdnjs`][cdnjs].
