How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/firefox_57-70`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/firefox_57-70`

* Use [`cdnjs`][cdnjs].
