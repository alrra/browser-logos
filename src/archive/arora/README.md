How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/arora`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/arora`

* Use [`cdnjs`][cdnjs].
