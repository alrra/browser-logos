How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/firefox_23-56`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/firefox_23-56`

* Use [`cdnjs`][cdnjs].
