## How do I get this logo?

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/samsung-internet-beta_5.4-9.1`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/samsung-internet-beta_5.4-9.1`

* Use [`cdnjs`][cdnjs].
