How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/internet-explorer-tile_10-11`
  * [`Yarn`][yarn]: `yarn add --dev @browser-logos/internet-explorer-tile_10-11`

* Use [`cdnjs`][cdnjs].
